# DIOR — Haute Couture E-Commerce Experience

A premium, luxury-grade e-commerce frontend designed to capture the elegance and sophistication of the **DIOR** brand. This project transforms a digital storefront into a high-end editorial experience, featuring bespoke haute couture collections, architectural boutique photography, and a seamless user interface.

## ✨ Features

- **Luxury Branding**: Fully realized Dior branding with high-end typography (`Playfair Display` & `Quicksand`) and a curated color palette (Burgundy, Soft Pink, and Classic White).
- **Designer Collection**: Exclusive "Designer" page featuring five unique haute couture gowns and four premium leather statement bags with bespoke descriptions and luxury pricing.
- **Dynamic UI**: Responsive grid layouts, smooth scroll-triggered reveal animations (using Intersection Observer), and a minimalist mobile-app aesthetic.
- **Audio Integration**: Immersive "cha-ching" sound effect upon completing a purchase for an enhanced user feedback experience.
- **Comprehensive Boutique Pages**:
  - **Home**: Featuring new arrivals and curated collections.
  - **Women's Collection**: A wide array of editorial clothes, bags, and accessories.
  - **Fragrances**: Dedicated showcase for iconic Dior scents.
  - **About Us**: Showcasing the architectural beauty of Dior's global boutiques.
  - **Checkout Flow**: Complete shopping bag, checkout, and confirmation sequence.

## 📂 Project Structure

The project is organized into a clean, modular directory structure:

```text
/
├── index.html          # Main landing page
├── designer.html       # Exclusive haute couture collection
├── women.html          # Women's fashion and accessories
├── fragrances.html     # Fragrance boutique
├── about.html          # Brand history and boutiques
├── cart.html           # Shopping bag overview
├── checkout.html       # Secure checkout experience
├── confirmation.html   # Order confirmation page
├── css/
│   └── style.css       # Core design system and animations
└── assets/
    ├── images/         # High-resolution editorial photography
    └── audio/          # Interactive sound effects (cash.mp3)
```

## 🛠️ Technologies Used

- **HTML5**: Semantic structure for accessibility and SEO.
- **CSS3**: Custom design system with Vanilla CSS (Flexbox, Grid, and Keyframe animations).
- **JavaScript**: Pure Vanilla JS for scroll reveals, filtering, and checkout logic.
- **Google Fonts**: `Playfair Display` (Serif) and `Quicksand` (Sans-Serif).

## 🚀 How to Run

1. Clone the repository to your local machine.
2. Open `index.html` in any modern web browser.
3. For the best experience, ensure your volume is up to hear the checkout sound effect!

---

*Note: This project is a frontend UI/UX demonstration. It does not include a backend for real payments or database persistence.*
